# intertwine-tutorial-project

## Instructions

### To Start App

1. Open a Terminal
2. 'nodemon watch' to run the app

### Create a lit-element using Cork

1. Type cork (then type Tab, that will autocomplete the command)
2. OR just type cork-app-utils
